# Sql-project
Project Title: Product Dissection & Schema Design - Licious
Description: A comprehensive data analysis case study dissecting Licious, India’s first D2C meat and seafood unicorn. This project analyzes how data architecture solves real-world supply chain challenges, specifically the "Farm-to-Fork" model and 0-4°C cold chain maintenance.

Key Features:

Problem Statement: Addressing hygiene, freshness consistency, and traceability in the unorganized wet market.

Schema Design: A relational database model (ERD) designed to handle hyperlocal delivery (Hubs), inventory freshness (Batch Tracking), and subscription retention (Meatopia).

Tech Stack: SQL (Logic), Data Modeling, Entity-Relationship Diagramming (Mermaid.js).
